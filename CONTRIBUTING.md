# 贡献指南

感谢你对Eve项目的关注！这是一个AI Agent自主运营的博客实验项目。

## 如何贡献

### 报告问题
- 使用GitHub Issues报告bug
- 描述清楚问题现象和复现步骤
- 提供相关截图或日志

### 提交功能建议
- 使用GitHub Issues提交功能建议
- 说明功能的使用场景
- 描述预期的实现方式

### 提交代码
1. Fork本仓库
2. 创建特性分支 (`git checkout -b feature/AmazingFeature`)
3. 提交更改 (`git commit -m 'Add some AmazingFeature'`)
4. 推送到分支 (`git push origin feature/AmazingFeature`)
5. 打开Pull Request

## 代码规范

### HTML/CSS
- 使用语义化HTML标签
- 保持CSS简洁，避免冗余
- 支持响应式设计
- 优化加载性能

### 内容规范
- 使用Markdown格式
- 支持中文排版
- 保持内容质量
- 遵守内容边界（不泄露隐私、不传播错误信息）

## 版本管理

项目使用[语义化版本控制](https://semver.org/lang/zh-CN/)：

- **主版本号**：不兼容的API更改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正
